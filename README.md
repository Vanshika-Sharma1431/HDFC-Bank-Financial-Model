# HDFC Bank Financial Model

## Overview
3-statement model (Income Statement, Balance Sheet, Cash Flow) and DCF valuation for HDFC Bank, built using FY23–25 standalone financial statements from HDFC Bank's annual reports. Model built in **Google Sheets** and exported as **Excel**.
## Key Findings
Two DCF methods were used with the same Cost of Equity **(14.23%)** for a fair comparison:
| Method | Intrinsic Value | vs. Market Price (₹775) |
|---|---|---|
| FCFE-based DCF | **₹165 per share** | -78.7% downside |
| Excess Return Method | **₹244 per share** | -68.5% downside |

> Both estimates fall significantly below the current market price of ₹799, suggesting the stock is trading at a **premium to intrinsic value** — as HDFC Bank's ROE (~11%) currently falls short of the 14.23% return shareholders require.

## Key Assumptions
- **Risk-Free Rate:** 6.85% (India 10Y G-Sec)
- **Beta:** 1.23 (Moneycontrol)
- **Equity Risk Premium:** 6.00% (Damodaran + India CRP)
- **Cost of Equity:** 14.23%
- **Terminal Growth Rate:** 8% (Sustainable Growth Rate = ROE × Retention Ratio)
- **Forecast Period:** FY26–FY30
